# snake-game-pygame
A classic Snake game developed in Python using Pygame with food, score, sound effects, and game over logic.
🐍 Snake Game using Python (Pygame)
📌 Project Summary

This is a classic Snake Game made using Python and Pygame.
The game is simple, fun, and beginner-friendly.
The snake eats food, grows in size, and the score increases.
If the snake hits the wall or itself, the game is over.

✨ Features

Snake movement using arrow keys

Food appears at random places

Score increases when snake eats food

Snake grows longer after eating food

Game Over when snake hits wall or itself

Restart game by pressing Enter

Sound effects for food and game over

Background image support

🎮 Controls

Up Arrow → Move Up

Down Arrow → Move Down

Left Arrow → Move Left

Right Arrow → Move Right

Enter → Restart game after Game Over

🛠 Technologies Used

Python

Pygame

📂 Project Files

snake_game.py → Main game file

bg.jpg → Background image

eat.wav → Food sound

gameover.wav → Game over sound

▶ How to Run the Game

Install Python

Install pygame using:

pip install pygame

Run the file:

python snake_game.py
